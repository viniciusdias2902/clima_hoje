# Clima Hoje

Um projeto de interface de previsão do tempo desenvolvido para praticar HTML e CSS, com foco em design responsivo e organização de código.

## Sobre o Projeto

Interface web que simula um aplicativo de previsão meteorológica, apresentando informações como temperatura atual, sensação térmica, chance de chuva, velocidade do vento e índice UV. O layout inclui previsão horária e semanal com ícones SVG customizados.

## Estrutura

```
.
├── index.html
├── style.css
├── mobile.css
└── images/
    ├── logo.svg
    ├── sun.svg
    ├── clouds.svg
    ├── rain.svg
    └── ...
```

## Características

- Design responsivo com breakpoint em 1024px
- Layout em grid para organização dos elementos
- Ícones SVG para diferentes condições climáticas
- Gradientes e sombras para profundidade visual
- Tipografia Poppins via Google Fonts
